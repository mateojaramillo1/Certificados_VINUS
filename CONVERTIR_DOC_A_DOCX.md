# Cómo Convertir .doc a .docx

Si el sistema no puede convertir automáticamente tu archivo `.doc` a `.docx`, sigue estos pasos:

## Opción 1: Microsoft Word

1. Abre tu archivo `.doc` en Microsoft Word
2. Ve a **Archivo** → **Guardar como**
3. En "Tipo de archivo" selecciona **Documento de Word (.docx)**
4. Haz clic en **Guardar**
5. Ahora puedes subir el nuevo archivo `.docx` al sistema

## Opción 2: Google Docs

1. Ve a [Google Docs](https://docs.google.com)
2. Haz clic en el icono de carpeta para abrir un archivo
3. Sube tu archivo `.doc`
4. Una vez abierto, ve a **Archivo** → **Descargar** → **Microsoft Word (.docx)**
5. Sube el archivo descargado al sistema

## Opción 3: LibreOffice (Gratis)

1. Descarga [LibreOffice](https://www.libreoffice.org/) si no lo tienes
2. Abre tu archivo `.doc` con LibreOffice Writer
3. Ve a **Archivo** → **Guardar como**
4. Selecciona **Microsoft Word 2007-2019 (.docx)** como tipo
5. Guarda y sube el nuevo archivo

## ¿Por qué es necesario?

Los archivos `.doc` usan un formato antiguo (binario) que no se puede modificar fácilmente.
Los archivos `.docx` usan un formato moderno (XML) que permite al sistema:
- Leer el contenido
- Reemplazar variables como `${nombre}`, `${cedula}`, etc.
- Generar certificados con los datos del empleado automáticamente

## Ventajas del .docx

✅ El sistema puede reemplazar automáticamente las variables
✅ Los empleados reciben certificados con sus datos ya completados
✅ No es necesario editar manualmente cada certificado
